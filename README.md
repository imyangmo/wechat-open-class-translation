## 2018 微信公开课张小龙演讲字幕翻译

### 参与流程

1. 向[掘金翻译计划管理员](https://github.com/leviding)提出申请

2. 加入掘金翻译计划·视频组微信群

3. 认领任务

4. Fork 本仓库

5. 翻译

6. 更新 README 中的译者信息

7. 提交并发 PR 到本仓库

8. 审核通过并 Merge
